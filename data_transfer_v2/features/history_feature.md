# History Page

This document describes the features of the History page, which is used for searching historical data records and exporting them.

## 1. Filtering and Searching

The page provides a form to filter historical data based on several criteria.

### Features:

-   **Start Date**: A date picker to select the beginning of the time range for the search.
-   **End Date**: A date picker to select the end of the time range.
-   **Car Plate**: An input field to search for a specific car plate number.
-   **Filter Button**: Applies the selected filters and displays the results.
-   **Export to Excel**: A button to export the filtered results as an Excel file.

## 2. Results Display

-   **Database Results**: The search results are dynamically loaded into a dedicated area on the page. While the exact format isn't specified in the template, it is expected to be a table or list of records matching the filter criteria.

## How to Use:

1.  **Set Filters**: Use the date pickers and the car plate field to define your search criteria.
2.  **Apply Filter**: Click the "Filter" button to fetch and display the matching historical records.
3.  **Review Results**: Analyze the data displayed in the results area.
4.  **Export Data**: If needed, click the "To Excel" button to download the filtered data for offline use.
