# Statistics Page

This document describes the features of the Statistics page, which provides a dashboard for visualizing data related to a service named "Kashif".

## 1. Filtering and Exporting

The page includes a form to filter the data used for the statistics and to export it.

### Features:

-   **Start Date**: A date picker to set the start of the reporting period.
-   **End Date**: A date picker to set the end of the reporting period.
-   **Car Plate**: An input to filter statistics for a specific car plate.
-   **Filter Button**: Applies the filters to the statistics and charts on the page.
-   **Export to Excel**: A button to export the filtered data to an Excel file.

## 2. Statistics Cards

A set of cards at the top of the page display high-level metrics.

### Key Metrics:

-   **Total Kashif Responses**: The total number of responses received from the Kashif service.
-   **Completed**: The number of successful or "correct" responses.
-   **Not Completed**: The number of failed or "fault" responses.

## 3. Charts

The page features several charts to provide a more detailed visual breakdown of the statistics.

### Available Charts:

-   **Kashif Responses**: A pie chart that likely shows the overall distribution of response types (e.g., success vs. failure).
-   **Wanted Status Code**: A chart visualizing the breakdown of "Wanted Status Codes" from the responses.
-   **Legal Status Code**: A chart visualizing the breakdown of "Legal Status Codes" from the responses.

## How to Use:

1.  **Set Reporting Period**: Use the date pickers to select the time frame you are interested in.
2.  **Filter (Optional)**: If needed, enter a car plate to narrow down the statistics.
3.  **Apply Filter**: Click the "Filter" button to update the stats and charts.
4.  **Analyze Data**: Review the key metrics on the cards and the detailed breakdowns in the charts.
5.  **Export**: Click "To Excel" to download the raw data for your filtered view.
